一份真正成熟可靠的组件化方案，需要具备的能力：

`“资源高内聚”`—— 组件资源内部高内聚，组件资源由自身加载控制

`“作用域独立”`—— 内部结构密封，不与全局或其他组件产生影响

`“自定义标签”`—— 定义组件的使用方式

`“可相互组合”`—— 组件正在强大的地方，组件间组装整合

`“接口规范化”`—— 组件接口有统一规范，或者是生命周期的管理
